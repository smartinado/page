Built by Tasmota power users for professional teams

Tasmagic started as an internal toolkit for Smartinado engineers who needed faster provisioning flows, safer Berry automation reviews, and modern dashboards for large Tasmota fleets. It has since become a fully supported commercial platform, with a perpetual demo build so evaluators can explore the experience anytime.

Mission

A single control room

Replace scattered tools with one delightful interface that covers discovery, provisioning, automation, analytics, and maintenance.

Principles

Offline-first & accountable

Everything runs locally, automation code is always reviewable, and commercial builds are maintained through a hardened release program.

Technology

Flutter + pure Dart

A single codebase powers Android, iOS, desktop experiments, and the documentation website you are reading now.

Product milestones

Discovery 2.0

Multi-network scans, MQTT health probes, and one-tap dashboard generation.

Automation Studio

Round-trip Berry parsing, trigger/condition diffing, and visual preview for complex flows.

Provisioning Suite

Captive portal assistant, Wi-Fi templates, and credential vault with secure sharing.

Insights & Backups

Historical sensor charts, log streaming, and encrypted device/automation backups.
